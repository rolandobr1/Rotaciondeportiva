# Preview Project Workflow

This workflow describes how to start the development server to preview the Court Commander project.

## Steps

1. **Start Development Server**
   Run the following command to start the Next.js development server:
   ```powershell
   npm.cmd run dev
   ```
   The server will start on [http://localhost:9002](http://localhost:9002).

2. **Access the App**
   Open your browser and navigate to `http://localhost:9002`.

3. **Genkit Development UI**
   If you need to work with Genkit AI flows, start the Genkit UI:
   ```powershell
   npm.cmd run genkit:dev
   ```
